Ignite CLI v29 installation:
=============================

brew install ignite (preferred)
or
sudo curl https://get.ignite.com/cli@v28.4.0! | bash


Scaffolding:
============

ignite scaffold chain esgobservabilitydemo
ignite scaffold list rawMaterialExtraction resourceType:string waterUse:string emissions:string --module esgobservabilitydemo
ignite scaffold list manufacturing componentType:string waterUse:string emissions:string --module esgobservabilitydemo
ignite scaffold list transportation transportationType:string fuelUse:string emissions:string --module esgobservabilitydemo
ignite scaffold list materialProcessing materialType:string waterUse:string emissions:string --module esgobservabilitydemo


ts-client Generation:
===========

ignite generate ts-client --clear-cache


Remote Deployment using Ignite App called Spaceship
=======================================================

ignite app install -g github.com/ignite/apps/spaceship
ignite app update github.com/ignite/apps/spaceship
ignite spaceship deploy ec2-user@13.233.107.70 --key $HOME/.ssh/id_rsa
